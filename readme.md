# Song SQL
This is an SQL program that I was just creating to mess around with making songs & Albums.


## Technology used 
- SQL
- SQLite3


## Setup/Install
### Install
```
git clone https://github.com/LWRGitHub/songs-sql.git
```

### Run
```
sqlite3 < songs_albums.sql
```
